# 分类导航 (Components)

Boring Design 提供了一套完整且实用的组件矩阵，按照功能和语义进行分类，以满足绝大多数业务场景的需求。

## 1. 基础组件 (General)

构建其他复杂组件和页面的基石。

*   [Button 按钮](./components/button/index.md)
*   Icon 图标
*   Typography 排版

## 2. 表单与数据录入 (Forms & Data Entry)

全面接入原生表单系统，支持序列化与校验。

*   Input 输入框
*   Textarea 多行文本
*   [Checkbox 复选框](./components/checkbox/index.md)
*   Radio 单选框
*   Select 选择器
*   Switch 开关
*   Slider 滑块
*   ColorPicker 颜色选择器
*   DatePicker 日期选择器

## 3. 反馈与提示 (Feedback & Overlays)

用于用户交互后的状态反馈与信息提示。

*   [Alert 警告提示](./components/alert/index.md)
*   Message 全局消息
*   Dialog / Modal 对话框
*   Drawer 抽屉
*   Toast / Snackbar 轻提示
*   Tooltip 文字提示
*   Popover 气泡卡片
*   Spinner / Skeleton 加载反馈

## 4. 数据展示 (Data Display)

用于结构化地展示复杂数据。

*   Card 卡片
*   Tag / Badge 标签与徽标
*   Avatar 头像
*   Tree 树形控件
*   Table 表格
*   Carousel 走马灯

## 5. 导航与布局 (Navigation & Layout)

提供页面骨架与视图切换能力。

*   Menu / Dropdown 下拉菜单
*   Tabs 标签页
*   Breadcrumb 面包屑
*   Split Panel 分割面板
*   Divider 分割线
